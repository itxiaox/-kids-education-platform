## 游戏机制

### 关卡结构
- 每关 10 道题
- 答对加分，答错生命值减少
- 通过需要 >= 70% 正确率

### 奖励系统
- ⭐ 完成关卡获得星星 (1-3颗)
- 🎁 连续答对3题额外奖励
- 🏆 完成阶段解锁成就

### 游戏状态
- 💗 生命值：3条命
- ⏱️ 计时模式
- 📊 分数统计

## 界面设计

### 主界面
- 冒险地图（可点击关卡）
- 进度显示（已完成关卡标记）
- 角色选择

### 游戏界面
- 问题区域（大字显示拼音/问题）
- 选项区域（4个选择按钮）
- 状态栏（生命值、分数、进度）
- 动画反馈（正确/错误）

### 结果界面
- 星级评定
- 分数统计
- 下一关/重试按钮

## 技术实现
- HTML5 Canvas 渲染
- JavaScript 游戏逻辑
- LocalStorage 存档
- CSS 动画效果
